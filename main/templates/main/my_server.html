Django Version

Since I needed admin pages and better backend management for Github integration, I decided to switch from more minimal library Flask to Django framework that has an ORM built in and decent admin interface.

The ThinkPad-T480 laptop with 8 CPU cores and 16GB of RAM was a overkill for my website. Thats why I changed to an old unused ASUS laptop and took the ThinkPad-T480 for personal use. Changing hardware was not a problem since I had already automated the django deployment with Ansible
